/*
Backup Types
____________
1) Full Database Backup = Full Backup (.bak)
	معمولا روزی یک بار خارج از وقت اداری
	سیستم های شلوغ هفته ای یکبار
	بدترین حالت ماهی یکبار
2) Transaction Log Backup = Log Backup (.trn)
	معمولا 90-15 ذقیقه یک بار
	نباید از 10-30 ثانیه بیشتر باشد زمان گرفتن
	خواندن کند نمیشود فقط رایت رو کمی کند میکند، این مشکل با دیسک دیجیتال حل میشود
3) Differential Backup = Diff Backup (.dif)
	تغییرات دیتا را ذخیره میکند از آخرین بکاپ کامل5

*/
